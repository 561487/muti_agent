1、基于requirement.txt安装对应的python环境。
2、配置human-in-the-loop及multi-agent里面对应的环境变量。
其中包含：
MODEL 模型名称，例如gpt-4o等。
MODEL_BASE_URL 调用模型的链接
MODEL_API_KEY 申请的API key
可自行部署大模型获取相关内容，也可调用公有云相关模型内容。
例如：https://www.aliyun.com/ 点击大模型进行对应key值申请。
也可上淘宝购买gpt-4o等模型的API中转

TAVILY_API_KEY 搜索接口API，可登录https://app.tavily.com/home进行申请
